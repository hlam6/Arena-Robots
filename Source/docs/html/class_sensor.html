Detailed Description

This class acts as an abstract interface for a family of sensors, which collect information.

Currently the only one implemented is SensorTouch. Sensor contains one private member variable, activated_, getters and setters for that variable, and another function called Accept() which should take an EventCollision as its argument. There is also a Reset() function, which should reset the object to its newly constructed state.
